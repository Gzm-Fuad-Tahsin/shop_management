# Shop Management System

This repository is split into two deployable apps:

- `frontend/`: Next.js frontend
- `backend/`: Express/MongoDB API

## Run Locally

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Backend:

```bash
cd backend
npm install
npm run dev
```

Set the frontend `NEXT_PUBLIC_API_URL` to the backend URL when deploying separately.
